### 项目概述
这是一个简单的 Web 应用，允许用户通过浏览器与 Grok AI 进行实时对话。用户需要提供自己的 API Key 来使用服务。



### 核心技术栈
- **前端**：HTML + CSS + JavaScript（原生）
- **后端**：Node.js + Express
- **通信**：Server-Sent Events (SSE)
- **API**：X.AI Grok API

### 主要功能
1. **API Key 管理**
   - 用户输入自己的 API Key
   - 本地存储 Key 信息
   - 密文显示保护隐私

2. **实时对话**
   - 发送消息给 AI
   - 流式接收 AI 响应
   - 实时显示对话内容

### 项目结构
```
project/
```

### 技术特点

### 使用方法
1. 获取 X.AI 的 API Key
2. 安装依赖：`npm install`
2. 启动应用：`node server.js`
3. 访问：`http://localhost:3000`
4. 输入 API Key 并保存
5. 开始与 AI 对话
